# Reactive

A React Native Boilerplate.

## Getting Started

Start your next React Native project in seconds

## Installing

```
git clone git@github.com:SMysakowski/Reactive.git
cd Reactive
yarn install
cd ios && pod install
```

```
yarn run ios
```

```
yarn run android
```
